Итоговое задание

Для выполнения итогового задания были выбраны: задание №7 Верстка по шаблону, задание №1 Регистрация участников конференции(первые 3 пункта задания) и задание №5 Тест по основам веб-программирования.

Верстка по шаблону была сделана на основе приложеной к заданию картинки. Макет состоит из 3 блоков (шапки, контента и футера), также есть общий вертикальный контейнер шириной 1080 пикселей.

Для выравнивания элементов страницы использовалась технология flex-box, для текста были изменены шрифты, в качестве заднего фона блоков были использованны картинки.

Были использованы псевдоклассы css, элементы с которыми можно взаимойдействовать при наведении на них мышкой меняют цвет шрифта или подчеркиваются.

Чтобы вернуться на главную страницу нажмите "ГЛАВНАЯ" в меню навигации.
